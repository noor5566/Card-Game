# Card Club — پتے کا کھیل

A multiplayer card game (Hearts & Hokm) built with Python + HTML.

## Games
- **Hearts** (دل) — Avoid collecting hearts and the Queen of Spades
- **Hokm** (حکم) — Trump card team game, first to 7 games wins

## Play Online
→ Deployed at: [your-app.railway.app]

## Run Locally
```bash
python server_final.py
# Open http://localhost:8766
```

## Deploy to Railway
1. Push this repo to GitHub
2. Go to railway.app → New Project → Deploy from GitHub
3. Select this repo → Deploy
4. Done! Share the URL with friends.

## Tech Stack
- Backend: Python 3 (stdlib only, no pip needed)
- Frontend: Single HTML file with vanilla JS
- Real-time: Raw WebSocket (no external libraries)
